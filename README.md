## advent of code
### structure
on the aoc About page, the creator asks that users please do not copy or share the aoc puzzles and inputs, making specific mention of not pushing them to a public repo.
i respect that, so the puzzles and inputs in this repo will be ignored by git, though i'll keep them on my local (and can always retrieve them again from the website if needed).
each year dir contains 25 day dirs.
each day dir contains a data dir to hold the puzzle and input (empty on the remote) and a work dir to hold the solutions
currently planning on solving these with some Lisp variant, but looking ahead I might want to solve them in other languages at too, so i'll probably subdivide the work dirs further by language.
